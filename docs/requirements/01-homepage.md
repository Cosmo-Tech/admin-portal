The home page of the portal admin is the organization view page of the main organization of the current tenant.

Most of the cases there is only one organization, so the user will land on the default organization after login.

This page has the standard navigation format (menus on the left, main body in the middle)

Menus:

Solutions => a click on this menu displays the Solutions List View showing the solutions of this organization

Workspaces => a click on this menu displays the Workspaces List View showing the workspaces of this organization
